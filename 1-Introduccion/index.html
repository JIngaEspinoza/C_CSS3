1. Como vincular HTML con CSS

Existen 4 métodos:

  • <link>
  • <style>
  • @import
  • style=""

<link>

La etiqueta link conecta al HTML con un recurso externo.

La sintaxis para conectar CSS es (recomendable en el HEAD)

    
<link rel="stylesheet" href="ruta/archivo.css">

En HTML5 no es neesario type="text/css"

<style>

La etiqueta <style> <style> permite embeber código css dentro de html

        
<style>
    body {
        background: red;
    }
</style>

@import

La regla @import añade una hoja de estilos externa dentro de otra.

Por lo tanto, si se usa en HTML, debe estar dentro de las etiquetas <style> </style> 

        
@import url(ruta/archivo.css)

style=""

El atributo <style> </style> permite insertar estilos en un elemento especifico de HTML

        
<h1 style="color: red">CSS3 Desde Cero</h1>

2. Sintaxis de CSS

  • regla (rule)
  • selector
  • declaración
  • propiedad
  • valor

regla (rule)

es un bloque de instrucciones css. omienza con un selector y termina on una llave de cierre.

        
selector {
    codigo css
}
@media {

}
@font-face {

}
@keyframes {

}

selector

Indican a que elemento HTML se va aplicar el estilo

    
p {
    parrafo
}

delaración (y bloque de declaración)

Una declaración es un estilo especifico. Y se conforma por una propiedad (que es el estilo a manipular ) y un valor que tomará dicho estilo

        
p {
    color: red;
}

* Nota1: El conjunto de todas las declaraciones encerradas entre llaves, se conoce como "bloque de declaraciones" (declaration block)

* Nota2: Tradicionalmente una propiedad tomaba un valor. Sin embargo en css3 existen varias propiedades que pueden tomar multiples valores

3. Selectores

  • Selectores de etiqueta
  • Selectores de clase
  • Selectores de ID
  • Selectores de descendientes
  • anidar selectores
  • selector de hijo directo

Selectores de etiqueta

Los estilos aplicados a las etiquetas se les conoce como estilos base.

        
body {
    background: grey;
}

Selectores de clase

Las clases son atributos de HTML que se indican con class="", estas se pueden manipular con CSS o JS.

Para utilizar una clase con CSS se utiliza un punto seguido del nombre de la clase.

        
HTML:
<p class="alerta"> Ha sucedido un error </p>

CSS:
.alerta {
    background: red;
    color: white;
}

Selectores de ID

Se selecciona un id con el simbolo #, seguido del nombre del id, sim embargo puesto que los Id son unicos no es reomendable usar Id en el CSS

    
HTML
<h2 id="titulo"> Curso desde cero </h2>

CSS
#titulo {
    color: yellow;
}

NOTA: 1 los Ids se usan para crear marcadores en el HTML o manipular el DOM on Js. Pero no se recomiendan para CSS. 2 Las clases se pueden usar con CSS o JS. Es buena practica nombrar con el prefijo js- a las clases de JS.

        
<ul class="menu" id="js-menu"></ul>

3. Las clases y los Ids puede unificarse en un selector. Esto significa que podrias seleccionar a elementos que tengas dos clases, o una clase y un ID, o una etiqueta y una clase, etc. 

    
.menu#header {
    background: green;
}

Selectores descendientes

Se le aplica a un elemento siempre que sea hijo de otro.

    
.padre .hijo {
    color: #fff;
}

* No importa el nivel del hijo

* Siempre se debe intentar descender lo menos posible

    
HTML:
<ul>
    <li>
        <a>Item 1</a>
        <a>Item 2</a>
    </li>
</ul>

CSS:
ul a {
    color: white;
}

Anidar selectores

Consiste en sepasar 2-selectores por comas, de esta manera los atributos se aplican a todos los selectores.

    
h1, p, a {
    font-size: 20px;
}

Selectores de hijo directo

Selecciona a los elementos que estan dentro directamento de otro elemento.

    
padre > hijo {
    color: lightblue;
}

4. Herencia

Significa que elementos ancestros heredan algunas propiedades a sus descendientes. Generalmente estilos como fuentes,colores, etc. Mientras los estilos de layaout por lo general no se heredan

    
HTML
<p>Soy un <span>parráfo</span></p>

CSS
p {
    color: red;
}

5.Cascada

La cascada viene del nombre de CSS (cascade style sheet) y significa que los estilos que vienen en ultimo lugar sobrescriben a los que estan antes.

    
p {
    color: red;
    color: green;
}

Se usa mucho la cascada para crear fallbacks para navegadores viejos.

    
p {
    background: red;
    background: linear-gradient(red,yellow);
}

Ten en cuenta que la cascada también se aplica cuando se inserta CSS con <link> o @import.

También se usa la cascada para simplificar los estilos.

    
.prev,
.next {
    background: red;
    color: white;
    float: right;
}
.prev {
    float: left;
}

6. Estilos de navegador

Todos los naveadores tienen sus estilos internos por defecto.

Es buena idea reescribir esos estilos.

    
h1 {
    font-size: 1em;
}

ul {
    padding-left: 0;
}

p {
    margin: 0;
}

a {
    text-decoration: none;
}

Podrias sobreescribir a mano los estilos. Pero existen librerias que ayudan mucho. Antiguamente se usaba reset css pero ya no es util.

La mejor opción es normalize
